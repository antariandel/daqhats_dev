This app is designed to operate with one MCC118.
With the use of a timed loop and thread, it will
read one sample from each channel, and display 
it on the form until.

It also includes a feature to flash the LED.

It was written using Code Blocks 16.1 and GTK+ 3.0

The project includes the IDE compiler and linker settings.
